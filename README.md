# Poe Ancients Price Helper RU

Оверлей для **Path of Exile 2**, который читает список наград на экране, получает актуальные цены с [poe.ninja](https://poe.ninja/poe2) и показывает цену рядом со строками списка.

Эта fork-сборка адаптирована под русский клиент PoE2: язык игры по умолчанию установлен в `Русский (Russian)`, а русские названия предметов переводятся в английские ключи poe.ninja через `locales/ru.json`.

## Что умеет

- Показывает живые цены poe.ninja рядом со строками списка.
- Учитывает количество предметов в строке, например `Сфера царей (3)`.
- Распознает русские названия валюты, рун, сплавов, расплавов и неограненных камней.
- Использует Windows OCR и Windows Graphics Capture, без Tesseract и внешних OCR-моделей.
- Хранит калибровку и настройки в `%LocalAppData%\PoeAncientsPriceHelper`, поэтому обновления их не стирают.
- Хранит app-managed fallback русской локализации в `%LocalAppData%\PoeAncientsPriceHelper\locales\_bundled`, чтобы автообновление не ломало уже рабочие русские названия.
- Ставит и обновляет приложение через GitHub Releases этого fork-репозитория.

## Установка

Скачай **`PoeAncientsPriceHelper-win-Setup.exe`** со страницы [Releases](../../releases) и запусти его.

Сборка самодостаточная, .NET runtime ставить отдельно не нужно. Windows SmartScreen может показать предупреждение, потому что приложение не подписано.

## Запуск

1. Запусти приложение.
2. Выбери лигу.
3. Нажми `Calibrate Region` или `F4` и выдели область списка наград в игре.
4. Нажми `Start`.

После калибровки приложение может стартовать сканирование автоматически и сворачиваться в трей. Это настраивается в `Settings`.

## Настройки

- `Game language`: для этой сборки по умолчанию `Русский (Russian)`.
- `Capture mode`: `Auto` использует GPU-захват с fallback на GDI, `Legacy (GDI)` принудительно включает старый захват.
- `Theme`: тема окна приложения.
- `Auto-start`: запускать сканирование и сворачивать в трей при старте.
- `Island Rumour helper`: экспериментальный помощник для слухов острова.

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `F4` | Заново выделить область списка |
| `F5` | Запуск / остановка сканирования |
| `F3` | Показать диагностические рамки |
| `Esc` / `Ctrl+Click` | Скрыть оверлей |

## Диагностика

Если строка не оценивается, открой `Diagnostics` или запусти приложение с `--debug`. Логи пишутся в `%LocalAppData%\PoeAncientsPriceHelper`.

Главные файлы для проверки:

- `scan_log.txt`
- `debug_ocr.png`
- `crash.log`

Если в логе строка помечена как `MISS`, для распознанного названия может не хватать алиаса в `src/PoeAncientsPriceHelper/locales/ru.json`.

## Сборка из исходников

Требуется .NET 10 SDK и Windows 10 2004+ / Windows 11.

```powershell
dotnet test src/PoeAncientsPriceHelper.Tests/
powershell -ExecutionPolicy Bypass -File tools/Build-Release.ps1
```

Готовые Velopack assets появляются в `release/`.
